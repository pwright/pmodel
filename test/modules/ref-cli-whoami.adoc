= rhoas whoami

[role="_abstract"]
ifdef::env-github,env-browser[:relfilesuffix: .adoc]

Print current username

[discrete]
== Synopsis

Print the username of the currently active user.

This command prints the username associated with the user currently logged in.


....
rhoas whoami [flags]
....

[discrete]
== Examples

....
# print current username
$ rhoas whoami

....
